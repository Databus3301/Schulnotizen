#### Beispiele

1

1) $f(x) = x³ -3x$
- $f'(x) = 3x² -3$
 $f''(x) = 6x$
2. NB: f'(x) = 0
- $3x² -3 = 0 | +3$
- $3x² = 3  | :3$
- $x² = 1 | \sqrt{}$
- $x_1 = 1$ ^ $x_2 = -1$
3. HB: f'(x) = 0 ^ f''(x) != 0

4. y-Wert
- $f(1) = -2 → T(1|–2)$
- $f(-1) = 2 → H(-1|2)$
